User guide

How to create a new Public Holiday?

  1. Go to Leaves > Configuration > Public Holidays.

  2. Click on create and fill in the form.

How to automatically remove the weekends from leave entries?

  1. Go to Leaves > Configuration > Settings > Leave days calculation.

  2. Check the boxes required.
