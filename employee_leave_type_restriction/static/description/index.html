Restrict Displaying Leave Types

The standard leave management application of Odoo allows showing all leave types to all the employees.
However, to display information to the user which is relevant to them only is one of the important features
to help improving better usability and security as well.

An effort towards that direction for leave management application, this module helps you to show leave
types to the employees which are configured for them.

This module helps to configure leave types in order to control its visibility toward various employees.

This module allows adding employee Tags in the Leave Types. Odoo will show all the leave types to the
employee by comparing leave type's tags with the Employee tags. If the leave type has no employee tags then
it will be shown to all the employees.
